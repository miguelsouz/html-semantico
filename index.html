TAGS ELEMENTO SEMÂNTICO - Importância

Conceito

Um elemento semântico descreve claramente seu significado tanto para o navegador quanto para o desenvolvedor. Muitos sites contêm código HTML, para indicar navegação, cabeçalho e rodapé.Em HTML existem alguns elementos semânticos que podem ser usados ​​para definir diferentes partes de uma página web

A Semântica HTML é importante para a indexação por mecanismos de busca, pois ajudam os robôs dos mecanismos de busca a entender melhor o conteúdo da página.

Algumas Tags

Conceito e demonstrações

Estas tags são chamadas também de elementos semânticos, ou seja, que conseguem passar uma informação com um significado específico para o conteúdo interpretado pelo navegador, não depende apenas do texto dentro da tag para se entender o que há naquela parte do site.

Exemplos de poucas tags

<Blockquote>

A tag especifica uma seção citada de outra fonte.Os navegadores geralmente recuam elementos

Exemplo:

Aqui está uma citação da Ford

Ford Motor Company é uma fabricante de automóveis multinacional estadunidense sediada em Dearborn, Michigan, um subúrbio de Detroit. Foi fundada por Henry Ford e incorporada em 16 de junho de 1903

<Figure>

A <figure> tag especifica conteúdo independente, como ilustrações, diagramas, fotos, listagens de códigos, etc.Embora o conteúdo do <figure7gt elemento esteja relacionado ao fluxo principal, sua posição é independente do fluxo principal e, se removido, não deverá afetar o fluxo do documento. Dica: O elemento <figcaption> é usado para adicionar uma legenda ao <figure> elemento
